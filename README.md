# Vitric

**The glass-box game engine for AI agents.**

现有引擎为"人坐在编辑器前"设计，对 AI 是黑盒；Vitric 以 agent API 为中心——引擎的一切状态对 AI 可见、可操作、可验证，AI 能自主地运行游戏、检测问题、迭代修改，闭环不需要人插手。

核心设计（详见 [docs/AI原生游戏引擎-设计稿.md](docs/AI原生游戏引擎-设计稿.md)）：

- **声明式数据层**：场景/实体/规则/素材槽全是强 schema 数据，写入即校验、运行时可往返读回
- **ECS 地基 + 规则正门**：80% 玩法由"当 X 则 Y"规则直译，复杂逻辑落 TypeScript 系统函数（声明读写范围）
- **AI 控制面**：引擎内建调试端口——截图、查状态、注入输入、时间控制、断言钩子，官方 MCP server
- **确定性 + 录像回放**：任何 bug 精确重放到出错前一帧，AI 调试从"猜"变成"看回放"
- **Rust + wgpu 运行时，2D 桌面 / Steam 发行**

状态：设计阶段。
