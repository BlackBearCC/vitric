# 回响 Echo — 完整单机立项稿 v0.1（导演:总纲,细化合同按章追加）

## 愿景一句话
定名《回响》:你布下的每一盏灯都是对黑暗的回应,残影是你自己的回响。
你是提灯人。影怪无法踏入光照之地——你的"牌"不是刀剑,是灯油、棱镜、遮板:
每一手牌都在重新雕刻这片黑暗的形状。光照是棋盘,阴影是敌人的路。

## 三层创新(全部引擎原生,别家做不出/没做过)
1. **光影即棋盘(主体)**:回合制网格战。影怪只能在阴影中移动/攻击,被光照住即僵直/受伤;
   玩家手牌=光源操作(放灯/移灯/棱镜分光/遮板挡光/燃料管理)。引擎的点光/聚光/阴影投射/法线
   是规则本体——挡光的墙(Solid)同时挡身体,物理与光学同一套数据。
2. **灯影回响(进阶,中期解锁)**:每场战斗自动按回合存快照。花"灯芯"资源可回溯 N 回合重打,
   但你上次的行动以"残影"形式按录像逐帧重演,残影的灯也真实照明——与过去的自己协同布光。
   确定性引擎的录像/快照是机制本体,残影分毫不差。
3. **低语者(彩蛋)**:某些影怪会说话(运行时 LLM,llm-ask),台词活的、行动严格走规则;
   断网/无 key 时按 llm-error 兜底为沉默怪,玩法完整性不依赖网络。

## 完整单机结构(证明引擎的部分)
主菜单(场景) → 章节地图(选路,3章×4-5战+章末Boss) → 战斗(网格光影回合制)
→ 战后奖励(三选一:新牌/灯具升级/灯芯) → 存档随时续玩 → 结局+制作名单。
预期时长 45-90 分钟,难度曲线由 QA 盲玩数据定。
